# AeviaLive — Creator Streaming Platform

Live streaming platform with real-time tips and creator monetization.

## Features

- HLS video streaming (hls.js, native Safari support)
- Real-time tips via WebSocket
- Creator dashboard with earnings tracking
- Live chat with moderation
- Mobile-optimized player

## Stack

NestJS · Next.js · Socket.io · hls.js · TypeScript
